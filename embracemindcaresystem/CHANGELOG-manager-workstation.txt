管理員專區 / 專業人員後台工作站更新

1. MainUI 保留為一般大眾第一個預約頁面。
2. MainUI 的「管理員專區」會開啟 ManagerAreaDialog。
3. ManagerAreaDialog 改為「請選擇您的身分」，只提供「醫師」與「心理師」兩個入口。
4. 醫師入口會開啟醫師身分登入：
   - 可下拉選擇醫師姓名
   - 測試帳號 doctor / 1234
   - 登入後進入醫師後台管理系統
5. 醫師後台新增：
   - 預約患者名單
   - 載入 MSE 範本
   - 外觀與行為總體描述
   - 多維度快選標籤矩陣
   - MSE 結構化病歷文字產生區
   - 用藥與症狀雙軸時間軸 UI 模擬
6. 心理師入口會開啟心理師身分登入：
   - 可下拉選擇臨床心理師 / 諮商心理師姓名
   - 測試帳號 psychologist / 1234
   - 登入後進入心理師後台管理系統
7. 心理師後台新增：
   - 預約患者名單
   - 左欄資料與素材庫
   - 轉介主訴與個案資料
   - 心理測驗分數輸入
   - 常態分配曲線圖
   - 雷達圖
   - 右欄結構化報告編輯器
   - 產生報告草稿、匯出 TXT、列印 / 另存 PDF
